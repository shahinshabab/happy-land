# 🖼️ 1. HERO / PROMOTION BANNER

### 📍 Location:

Top section (main banner)

### ✅ Recommended size:

```
1920 × 800 px  (BEST)
```

### Alternative:

```
1600 × 700 px (OK)
```

### ❌ Avoid:

* Square (1980×1980 ❌)
* Too tall images

### 📝 Text guideline:

* Keep text minimal (or no text)
* Example:

  * "Ramadan Sale"
  * "Special Offers"
  * "Wholesale Prices Available"

### 📂 Folder:

```
assets/homepage/hero/
```

# ⚡ PERFORMANCE TIP (VERY IMPORTANT)

Before uploading images:

### Compress images using:

* [https://tinypng.com](https://tinypng.com)
* [https://squoosh.app](https://squoosh.app)

### Target sizes:

* Hero: < 300KB
* Banners: < 250KB
* Products: < 100KB
* Icons: < 30KB

---

# 🎯 PRO DESIGN RULES

✔ Same style images per section
✔ Same background color (white or clean)
✔ Avoid mixed lighting
✔ Keep products centered
✔ Use high resolution (no blur)
